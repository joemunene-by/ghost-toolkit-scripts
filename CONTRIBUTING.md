# Contributing to Ghost Toolkit

Welcome! This guide will help you contribute to Ghost Toolkit.

## 🤝 How to Contribute

### 1. Report Bugs
- Use GitHub Issues to report bugs
- Include steps to reproduce
- Specify your environment (OS, tools installed)

### 2. Suggest Features
- Open a GitHub Issue with your idea
- Explain the use case
- Provide examples

### 3. Add New Tools
- Create wrapper scripts in `tools/<category>/`
- Follow existing script patterns
- Include help flag (`-h`, `--help`)
- Add to launcher menu

### 4. Improve Documentation
- Fix typos
- Add examples
- Translate to other languages

## 📝 Pull Request Guidelines

1. **Fork** the repository
2. **Create** a feature branch: `git checkout -b feature/amazing-tool`
3. **Commit** your changes: `git commit -m "Add: Amazing tool"`
4. **Push** to your fork: `git push origin feature/amazing-tool`
5. **Open** a Pull Request

## � Coding Standards

- Use 4-space indentation
- Add comments for complex logic
- Include error handling
- Test your script before submitting
